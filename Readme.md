# Start learning variables

```python
x, y, z = "Orange", "Banana", "Cherry"
print(x, y, z)

# x = "Mango"
x = str(3)     # Casting to string
y = int(3)     # Casting to integer
z = float(3)   # Casting to float

print(x, y, z)

# Checking the type of variable using type() function
print(type(x))
# End of variable section

```
